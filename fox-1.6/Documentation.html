Type display description

None

Это свойство скрывает элемент полностью и удаляет его из потока документа. Это означает, что элемент не занимает места на странице, и он не будет виден пользователю.

Block

Это свойство делает элемент блочным, что означает, что элемент будет занимать всю доступную ширину на странице и будет располагаться под предыдущим элементом.

Flex

Это свойство делает элемент контейнером для flexbox-элементов. Это означает, что элемент будет управлять поведением и расположением своих дочерних элементов, используя свойства flexbox.

Grid

Это свойство делает элемент контейнером для grid-элементов. Grid-элементы используются для создания сеток, которые могут быть настроены для управления размещением и размерами элементов внутри сетки.

Subgrid

Это свойство позволяет элементу-потомку наследовать сеточные линии из своего родительского элемента, когда он находится внутри grid-контейнера, который использует свойство grid-template-rows или grid-template-columns.

Contents

Это свойство преобразует элемент в контейнер, который содержит только контент, расположенный внутри элемента, и не включает в себя отступы, границы или фон.

Flow-root

Это свойство устанавливает блочный контейнер, который создает новый блочный форматированный контекст и содержит дочерние элементы в этом контексте. Это свойство может быть полезно для решения проблем с переполнением контента и позиционированием.

List-item

Это свойство делает элемент контейнером для элементов списка, таких как li. Это свойство обычно используется в сочетании со свойством list-style для создания списка с маркерами.

Run-in

Это свойство определяет, как элементы будут взаимодействовать друг с другом, когда они находятся на одной строке. Если элементы имеют свойство display: run-in, они будут встраиваться в контекст своих соседних элементов, если это возможно.

Inherit

Это свойство наследует значение свойства display у родительского элемента.

Initial

Это свойство устанавливает начальное значение свойства display для элемента, которое зависит от типа элемента. Например, для большинства элементов, значение свойства initial будет display: inline.

Unset

Это свойство сбрасывает значение свойства display элемента на его исходное значение. Если у элемента было задано значение display через CSS-правило, то свойство unset сбросит это значение на исходное значение, которое задано в стандарте для этого элемента.

Table

Это свойство делает элемент блочным контейнером для таблицы, который содержит один или несколько элементов table-row. Он может также содержать элементы table-caption, table-column-group, table-header-group и table-footer-group.

table-caption

Это свойство делает элемент внутри контейнера table заголовком таблицы, который располагается над самой таблицей.

table-column

Это свойство делает элемент-контейнер, который представляет одну колонку таблицы. Он используется для управления шириной колонки и ее свойствами.

table-column-group

Это свойство делает элемент-контейнер, который представляет группу колонок таблицы. Он используется для управления шириной нескольких колонок и их свойствами.

table-row

Это свойство делает элемент-контейнер, который представляет одну строку таблицы. Он может содержать один или несколько элементов table-cell

table-row-group

Это свойство делает элемент-контейнер, который представляет группу строк таблицы. Он используется для управления свойствами группы строк таблицы.

table-header-group

Это свойство делает элемент-контейнер, который представляет заголовки таблицы. Он содержит одну или несколько групп строк таблицы.

table-footer-group

Это свойство делает элемент-контейнер, который представляет нижние части таблицы. Он содержит одну или несколько групп строк таблицы.

table-cell

Это свойство делает элемент-контейнер, который представляет одну ячейку таблицы. Он должен быть вложен в элемент table-row. Он может содержать любой контент, включая другие элементы и таблицы.

Inline

Это свойство делает элемент инлайновым, что означает, что он может быть расположен в строке с другими элементами. Элемент inline не может иметь высоты и ширины и должен быть по размеру своего контента.

Inline-block

Это свойство делает элемент инлайновым блоком, который может быть расположен в строке с другими элементами и иметь свойства блочного элемента, такие как высоту, ширину и поля.

Inline-flex

Это свойство делает элемент инлайновым элементом, который использует гибкие свойства расположения, как в элементе flex.

Inline-grid

Это свойство делает элемент инлайновым элементом, который использует сетку для расположения содержимого.

Inline-table

Это свойство делает элемент инлайновым элементом, который представляет собой таблицу, но может быть расположен в строке с другими элементами.

Inline-box

Это свойство делает элемент инлайновым элементом, который использует свойства боксового элемента, как в элементе box. Это свойство не является частью стандартной спецификации CSS и не поддерживается всеми браузерами.

Ruby

Это свойство позволяет определить Ruby-разметку для элемента. Оно делает элемент родительским для Ruby-разметки, которая состоит из параллельных фрагментов текста, размещенных над основным текстом, и используется для указания фонетической пометки для сложных символов. Это свойство задает контейнер, который содержит как ruby-base-container, так и ruby-text-container.

Ruby-base

Это свойство задает текст, который используется в основном содержании элемента Ruby-разметки. Оно указывает на текстовый фрагмент, для которого будет добавлена руби-надпись.

Ruby-base-container

Это свойство задает контейнер, который содержит ruby-base. Это свойство определяет элемент, содержащий текст, для которого будет добавлена руби-надпись.

Ruby-text

Это свойство задает текст, который будет использоваться для написания руби-надписи в Ruby-разметке. Оно указывает на текст, который будет добавлен над основным текстом.

Ruby-text-container

Это свойство задает контейнер, который содержит ruby-text. Оно определяет элемент, содержащий текст, который будет использоваться для написания руби-надписи.
